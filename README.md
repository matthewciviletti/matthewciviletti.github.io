Creative Commons Attribution-NonCommercial 4.0 International Licence.

Live site:
https://matthewciviletti.github.io

## Structure

- index.html – homepage
- menu.html – shared navigation
- footer.html – shared footer
- css/styles.css – site styling
- images/ – images used on the site

## Accessibility

I have designed this site to follow WCAG 2.2 Level AA accessibility guidelines.
